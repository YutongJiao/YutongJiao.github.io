Practice

With the Electronic medical Records(EMR), clinical staff need to fill out the patient encounter form every time when they see a patient. The doctors are busy with large number of appointments or walk-in patients every day. Hence, we are looking to find the solutions to reduce the doctors' workload and help save time. Moreover, it is not convenient for doctors to fill out the encounter form with portable mobile app while seeing patients outside. In order to help clinical staff save time and increase the efficiency of running the EMR, we are considering to integrate automated features into the EMR, such as automated diagnostic code assigning, fee code assigning and dictation transcription. With the rapid rising up of Amazon Web Services, we consider using the services to perform these tasks and integrate them with our EMR..

Frontend Web Development and Maintenance

1. Create a new Lightsail instance for VR Concussion Testing hosted in AWS Route53 and develop the website with WordPress.
2. Renew Let’s Encrypt SSL certificates for nine AWS Lightsail websites and three EC2 instances.
3. Archive the detailed renewal tutorials and tips for Lightsail instance and EC2 instance.

Automated Diagnostic Code Extraction

1. Run VEMR virtual server on AWS EC2 as an instance

Run MySQL database in the server to store the patients’ information and documents in VEMR, including dictation audios from clinical doctors to diagnose patients.

2. Transcribe the dictation audio from the VEMR

(1) Create transcription job using AWS Transcribe Medical.
(2) Retrieve the transcription text to VEMR MySQL database.
(3) Back up dictation audio to AWS S3 Bucket.

3. Build AWS Lambda function for clinical text processing and analysis

(1) Extract diagnosis code (ICD-10-CM) from transcription text using AWS Comprehend Medical.
(2) Back up transcription text to AWS S3 Bucket.

4. Configure RESTful API with integration type of Lambda function

(1) Create “PUT” method for the API to send doctor note, namely transcription text to the Lambda function.
(2) Create “GET” method to retrieve the diagnosis code from Lambda function.
